Kinase search:

 *The kinases can only be searched using Kinase Uniprot ID, Kinase Gene Name, Alias Name.



Inhibitor search:

 *The inhibitors can also be searched using Inhibitor ChEMBL ID, Inhibitor Name, Kinase Targets.

Phosphosite search:

 *The phosphosites can be searched using the substrate name which are in Substrate Gene Name, Substrate Uniprot ID, Substrate Name.


Phosphoproteomic data upload:

 The choose file only supports .tsv file format.
